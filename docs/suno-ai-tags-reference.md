### üéµ Song Structure Tags
Primary structure tags that define song sections:
- `[Intro]` - Opening instrumental or vocal section
- `[Verse]` / `[Verse 1]` / `[Verse 2]` - Main lyrical sections
- `[Pre-Chorus]` - Build-up before chorus
- `[Chorus]` - Main hook/repeated section
- `[Post-Chorus]` - Follow-up to chorus
- `[Bridge]` - Contrasting section
- `[Outro]` / `[Outro, fade]` - Ending section
- `[Hook]` - Catchy repeated phrase
- `[Break]` / `[Instrumental Break]` - Instrumental pause
- `[Interlude]` - Musical transition
- `[Refrain]` - Repeated line or phrase
- `[End]` / `[Fade Out]` / `[Fade In]` - Explicit end markers

## üí° Usage Guidelines

### Best Practices
1. **Tag Placement**: Place structural tags at the beginning of lines
2. **Combination Limit**: Use maximum 3-4 tags simultaneously
3. **Clear Syntax**: Always use square brackets `[]`
4. **English Only**: All tags must be in English
5. **Early Placement**: Tags work best in first 20-30 words

### Example Song Structure
```
[Intro] [Soft Piano] [BPM: 75]
[Verse 1] [Female Vocal] [Gentle]
Lyrics here...

[Pre-Chorus] [Build Up] [Harmonies]
Building tension...

[Chorus] [Powerful] [Catchy Hook] [Energy: High]
Main hook here...

[Verse 2] [Male Vocal] [Emotional]
Second verse...

[Bridge] [Orchestral] [Crescendo]
Contrasting section...

[Outro] [Fade Out] [Ambient Wash]
```

### Interactive Editor Features
For the drag-and-drop interface, each tag category should be:
- **Draggable**: Users can drag tags into lyrics
- **Contextual**: Show relevant tags based on section
- **Visual**: Color-coded by category
- **Validated**: Check tag compatibility
- **Previewable**: Show effect description on hover

- **Section Headers**: For structural tags

### Tag Validation Rules
- Structural tags cannot overlap
- Certain tags are mutually exclusive (e.g., [Male Vocal] vs [Female Vocal])
- Some tags require parameters (e.g., [BPM: xxx])
- Platform tags should be used sparingly
